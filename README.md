# Power-Bi-Tasty-Chocolate-Company-Sales-Analysis
End-to-end Power BI business analysis project using shipment, product, people and calendar data to calculate profit, sales trends, shipment distribution and top performing salespersons.

## Project Overview
This project analyzes the sales performance of a chocolate company using Power BI.
The dashboard provides insights into revenue trends, shipment activity, product performance, and salesperson profitability.

The dashboard provides insights about:
- Sales trends
- Profit analysis
- Shipment distribution
- Top performing salespersons
- Product performance
- Geo sales distribution

## Dataset
The project uses four tables:

1. Shipments – Contains shipment sales data
2. Products – Contains product details
3. People – Contains sales team information
4. Calendar – Used for time intelligence analysis

## Data Model 
Relationships created:

Shipments[Product_ID] → Products[Product_ID]

Shipments[Sales_Person] → People[Sales_Person]

Shipments[Date] → Calendar[Date]

## DAX Measures
1. Total Selling Amount = SUM(Shipments[Amount])
2. Total Cost Amount = SUM(Shipments[CostPrice])
3. Total Profit Amount = [Total Selling Amount] - [Total Cost Amount]
4. Profit % = DIVIDE([Total Profit Amount], [Total Selling Amount])
5. Shipment Count = COUNTROWS(Shipments)
6. Total Boxes = SUM(Shipments[Boxes])
7. Amount per Box = DIVIDE([Total Selling Amount], [Total Boxes])
8. Total Amount (Last Year) = CALCULATE([Total Selling Amount], SAMEPERIODLASTYEAR(Calendar[cal_date]))
9. Amount Variance =
VAR amt_lstyr = [Total Amount (Last Year)]
RETURN
IF(
    ISBLANK(amt_lstyr),
    BLANK(),
    [Total Selling Amount] - amt_lstyr
)

## Dashboard Visualizations
The Power BI dashboard contains several visualizations that help analyze business performance.

Profit Analysis
Shows profit generated by products and salespersons.

Shipment Trend Analysis
A line chart showing shipment count by month to understand shipment trends over time.

Amount per Box Analysis
Evaluates revenue efficiency by calculating the average amount generated per box by each salesperson and across regions.

Top & Bottom Salespersons
Identifies the highest and lowest profit-generating salespersons.

Sales Trend (Current Year vs Previous Year)
Compares monthly sales with the same month in the previous year.

Geo Sales Distribution
Displays revenue contribution from different countries.

## Business Insights
From the dashboard analysis, several important insights were identified:

1. A small group of salespersons generates a large portion of total profit.
2. Sales performance shows clear fluctuations across months.
3. Some regions contribute significantly higher revenue compared to others.
4. Revenue efficiency varies between salespersons when measured by amount per box.
5. Shipment distribution highlights operational workload patterns.

These insights help the company improve sales strategy, resource planning, and profitability management.
